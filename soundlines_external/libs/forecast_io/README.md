# forecast.io.rs

An unofficial Rust wrapper for the [forecast.io](https://forecast.io) API.


### Installation

Add the following dependency to your Cargo.toml:

```toml
forecast_io = "0.3"
```

And include it in your project:

```rust
extern crate forecast_io;
```
